## 画面設計

### 自分の投稿一覧ページ

- 出入力

| 出入力 | 項目         |
| ------ | ------------ |
| 出力   | 投稿タイトル |
| 出力   | 投稿作成日   |

<!--   | 出力         | 投稿ユーザー | -->

- アクション
  | 項目 | 種類 | 働き |
  | ---- | ---- | ----|
  | 投稿作成 | リンク | 投稿作成ページに遷移する |
  |投稿編集 | リンク | 該当の投稿の編集ページに遷移する |
  | 投稿削除 | ボタン | 確認ダイアログを表示し、OKなら該当の投稿を削除して一覧ページに戻る |

### 投稿作成ページ

- 出入力
  | 出入力 | 項目 |
  | ---- | ---- |
  | 入力 | 投稿タイトル |
  | 入力 | 投稿内容 |

- アクション
  | 項目 | 種類 | 働き |
  | ---- | ---- | ----|
  | 保存 | ボタン | 入力内容をバリデーションし、問題なければ投稿を保存して一覧ページに遷移する |
  | キャンセル | リンク | 入力内容を破棄して一覧ページに戻る |

### 投稿編集ページ

- 出入力
  | 出入力 | 項目 |
  | ---- | ---- |
  | 出力 | 投稿作成日（編集不可） |
  | 入力 | 投稿タイトル（既存の値を初期表示） |
  | 入力 | 投稿内容（既存の値を初期表示） |

- アクション
  | 項目 | 種類 | 働き |
  | ---- | ---- | ----|
  | 更新 | ボタン | 入力内容をバリデーションし、問題なければ投稿を更新して一覧ページに遷移する |
  | キャンセル | リンク | 編集内容を破棄して一覧ページに戻る |

## URL設計

| URL                | メソッド | 処理                         |
| ------------------ | -------- | ---------------------------- |
| /posts             | GET      | 投稿一覧ページを表示する     |
| /posts/create      | GET      | 投稿作成ページを表示する     |
| /posts/create      | POST     | 投稿作成処理を実行する       |
| /posts/{id}/edit   | GET      | 投稿編集ページを表示する     |
| /posts/{id}/edit   | POST     | 投稿編集処理を実行する       |
| /posts/{id}/delete | GET      | 投稿削除確認ページを表示する |
| /posts/{id}/delete | POST     | 投稿削除処理を実行する       |

## データベース設計

### 投稿（posts）テーブル

| カラム論理名 | カラム物理名 | 型           | 型の意味            |
| ------------ | ------------ | ------------ | ------------------- |
| ID           | id           | INTEGER      | 連番（自動採番）    |
| タイトル     | title        | VARCHAR(100) | 100文字までの文字列 |
| 内容         | content      | TEXT         | 長文の文字列        |
| 作成日       | created_at   | TIMESTAMP    | 日付と時刻          |
| 更新日       | updated_at   | TIMESTAMP    | 日付と時刻          |
